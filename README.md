**Deployment info:**
- Create a .env file on the root directory (/backend/) including the following information:
```
DB_USER = <db username>
DB_PASS = <db password>
DB_DIALECT = <sequelize dialect: mysql, postgres, etc>
DB_HOST = <db host>
DB_DATABASE = <db database>
PORT = 3001
```
- npm install && npm start
